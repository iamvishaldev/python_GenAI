2️⃣ Boolean in Python

Boolean has only two values:

True
False
Important Insight

True = 1

False = 0

Example:

print(5 + True)   # 6
print(10 + False) # 10

3️⃣ Comparison Operators

These return Boolean values.

x = 10

print(x > 5)   # True
print(x == 10) # True
print(x != 3)  # True

Operators:

>   <   >=   <=   ==   !=

4️⃣ Logical Operators
AND

Both conditions must be True.

True and False  # False
OR

At least one True.

True or False  # True
NOT

Reverses the value.

not True  # False

5️⃣ Type Conversion to Boolean
bool(0)      # False
bool(10)     # True
bool("")     # False
bool("chai") # True

Rule:

Empty → False

Non-empty → True

🧠 Key Learnings Today
